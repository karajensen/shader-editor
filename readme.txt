Scene for editing shaders in real-time.

C++, Boost, Qt, DirectX, OpenGL,

• Real time shader editing
• Shader fragment linker
• Shader assembly output
• Multithreaded Qt GUI
• PyQt Maya exporter
• Colour correction
• Switching between
